## 🎯 什麼情境該想到我
當網站/頁面「載入很慢、首屏遲遲不出來」，你想知道從哪裡下手優化時。

## ⚙️ 怎麼用
> 黃金法則：**多數載入時間在前端** → 先優化前端（比調後端 CP 值高）。核心是**減少往返次數**。
1. **減少 HTTP 請求**：合併 JS/CSS、用 CSS sprites、行內小圖（越少往返越快）。
2. **JS 放底部、CSS 放頂部**：避免腳本阻塞頁面渲染。
3. **壓縮傳輸**：開 Gzip/Brotli，縮小 JS/CSS/圖片。
4. **用 CDN**：靜態資源就近提供，降低延遲。
5. **先量測再優化**：用瀏覽器開發者工具/效能瀑布圖找真正的瓶頸資源。
